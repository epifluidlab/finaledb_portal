## Startup

```
git clone https://bitbucket.org/zhu1lx/express_react.git
```

Install and update missing dependencies
```
pip install -r requirements.txt
yarn
cd client
yarn
```

Start server
```
# in `/cfdna_frontend`
yarn start:dev
```

Start frontend
```
# in `/cfdna_frontend/client` in new terminal tab
yarn start
```
